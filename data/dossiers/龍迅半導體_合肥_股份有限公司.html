一、这是个什么生意

龙迅半导体是一家芯片设计公司，但它自己不建厂生产，这种模式叫 fabless（无晶圆厂）【招股书 p.10】。

它主要干两件事：让数据传得更快、让视频显示得更清楚。

1. 智能视频芯片：负责处理视频信号，比如把电脑的HDMI信号转成手机屏幕能识别的MIPI信号，或者把4K/8K的高清视频“翻译”给显示器看【招股书 p.10】。

2. 互连芯片：负责在电脑、服务器内部，让CPU、GPU、内存这些核心部件之间高速、稳定地交换数据，解决信号传输中的衰减和干扰问题【招股书 p.10】。

这些芯片最终用在四个地方：

  • 智能视觉终端（占比最大，2025年收入占76.1%）：比如无人机、机器人、智能会议系统、商业显示器【招股书 p.11】。
  • 智能车（2025年收入占15.7%）：用在汽车的智能座舱和自动驾驶系统里，连接摄像头、雷达和显示屏【招股书 p.11】。
  • AR/VR眼镜（2025年收入占6.4%）：让眼镜能显示高分辨率、高刷新率的画面，同时保持低延迟【招股书 p.11】。
  • AI与高性能计算（2025年收入占1.8%）：为AI服务器和数据中心开发更高速的互连芯片，这是他们正在拓展的新方向【招股书 p.11】。

简单说，龙迅就是做各种电子设备里负责“传数据”和“显图像”的桥梁芯片。

二、赚不赚钱

很赚钱，但利润率在微降。

  • 毛利率：2023年53.6% → 2024年54.3% → 2025年52.9%，保持在50%以上，但2025年略有下滑【招股书 p.14】。
  • 净利润率：2023年31.8% → 2024年31.0% → 2025年30.3%，虽然很高，但也是逐年小幅下降【招股书 p.14】。

钱花在哪了？

  • 成本大头是晶圆（就是芯片的原材料），占销售成本的65%以上【招股书 p.15】。
  • 销售靠经销商：98%以上的收入都是通过经销商卖出去的，自己直接卖的很少【招股书 p.13】。

三、过去怎么走的，未来怎么说的

过去怎么走的（增长路径）

  • 2006年成立，最初做高清视频接口芯片【招股书 p.85】。
  • 2020年，进入智能汽车领域【招股书 p.85】。
  • 2023年，在A股科创板上市，同时开始在AR/VR领域取得进展【招股书 p.85】。
  • 2025年，开始拓展AI & HPC市场【招股书 p.85】。
  • 增长引擎：过去几年，智能车业务增长最快（2023-2025年复合年增长率59.6%），AR/VR和智能视觉终端也保持20%以上的增速【招股书 p.11】。

未来怎么说的（战略规划）

公司计划在招股书里说了五个方向【招股书 p.101】：

2. 瞄准高价值场景：重点攻智能车、AR/VR和AI & HPC这三个市场。

3. 拓展海外市场：计划在海外建研发中心，和更多国际芯片巨头合作。

4. 加强人才：招揽顶尖人才，搞股权激励。

5. 投资并购：看准机会，收购一些有技术的小公司。

四、行业什么样

  • 增长最快的细分领域：视频桥接芯片（智能视频芯片里的一种）和AI运力芯片（互连芯片里的一种）是未来增长最快的两个方向【招股书 p.62】【招股书 p.64】。
  • 竞争格局：

* 视频桥接芯片：龙迅在这个细分领域里，按2025年收入算，是中国内地排名第一的fabless设计公司，全球市场份额为4.1%【招股书 p.10】。但全球老大是德州仪器（美国），占了36.4%的份额【招股书 p.66】。

  • 行业特点：技术门槛高、客户认证周期长、供应链稳定要求高。新玩家很难进来【招股书 p.67】。

五、生意里的数字反差

3. 经销商依赖与客户集中的反差：公司98%以上的收入靠经销商【招股书 p.13】，但前五大客户就占了总收入的52.3%【招股书 p.14】。虽然客户分散在经销商手里，但收入来源其实高度集中在少数几个大经销商身上。

4. 供应商依赖：前五大供应商占了采购总额的90.6%，其中最大的一家就占了39%【招股书 p.14】。供应链风险高度集中，万一这家供应商出问题，生产就会受影响。

六、值得关注的点 + 待核清单

值得关注的点

2. AR/VR的生态卡位：公司已经和索尼、高通等巨头合作，芯片被用到索尼的AR显示面板参考设计中，并被雷鸟、Rokid等品牌采用。在AR/VR这个未来赛道里，龙迅已经占了一个不错的位置。

3. 国产替代的逻辑：在视频桥接芯片领域，龙迅是国内第一，正在挑战海外巨头的地位。在国产化替代的大背景下，这是一个明确的增长逻辑。

4. 财务非常健康：公司几乎没有负债，现金充裕，有足够的弹药去支撑研发和可能的并购。

待核清单

1. [缺出处·待核] 招股书提到“按2025年收入计，在视频桥接芯片市场，我们为中国内地排名第一的fabless设计公司”，但未明确说明这个“中国内地排名第一”是仅指本土公司，还是包含了在中国有业务的外资公司。需要确认排名口径。

2. [缺出处·待核] 招股书提到公司已与英伟达、高通、英特尔等有参考设计合作，但未披露合作的具体深度、排他性条款以及是否已产生实质性收入。需要进一步了解这些合作对业绩的实际贡献。

3. [缺出处·待核] 公司2025年收入增速（21.9%）相比2024年（44.2%）明显放缓，招股书未解释具体原因。需要关注是行业周期问题还是公司自身竞争力问题。

4. [缺出处·待核] 智能视频芯片的平均售价从2023年的15.4元/颗降到2025年的13.2元/颗，公司解释是低价成熟产品卖得多了。需要关注这个趋势是否会持续，以及是否会进一步拉低整体毛利率。

5. [缺出处·待核] 公司提到正在研发PCIe 5.0 Retimer，但未披露预计的流片时间、量产时间以及目标客户。需要评估该产品的商业化前景和风险。
