【拆解档案】弥费科技（上海）股份有限公司

一、这是个什么生意

弥费科技是一家给半导体晶圆厂做“快递系统”的公司。

1. 核心产品：自动物料传送系统（AMHS）

  • 干什么用的： 在芯片制造工厂里，有上千道工序，晶圆（做芯片的圆盘）需要在不同的机器之间跑来跑去。弥费科技做的就是一套软硬件结合的自动化系统，让天车、机器人、智能仓库自己完成搬运，不用人动手【招股书 p.25】。
  • 为什么重要： 这套系统一旦出问题，整个晶圆厂的生产线就可能停摆，损失巨大【招股书 p.25】。它直接影响芯片的产量、设备利用率（稼动率）和良率【招股书 p.3】。

2. 产品具体长什么样？

  • 硬件设备（三大类，三十多款）【招股书 p.3】：

- 传送设备： 主要是“天车”（OHT），在工厂天花板轨道上跑，负责把晶圆盒快速、平稳地送到指定机器【招股书 p.93】。

- 存储设备： 像智能立体仓库（Stocker），用来临时存放晶圆盒，方便随时取用【招股书 p.94】。

- 净化设备： 给晶圆盒吹干净的气体，防止芯片被灰尘或化学物质污染，保证良率【招股书 p.96】。

  • 软件系统（大脑）：

- 物料控制系统（MCS）： 整个系统的“中央神经中枢”，负责给所有硬件下达指令，规划最优路线【招股书 p.96】。

- 天车智能调度系统（MSTC）： 专门指挥天车集群，避免“堵车”，让搬运效率最高【招股书 p.97】。

3. 怎么赚钱？

  • 卖项目： 给晶圆厂提供一整套AMHS解决方案（整厂订单），包括所有硬件和软件【招股书 p.92】。
  • 卖设备： 单独卖存储设备、净化设备等单项硬件【招股书 p.98】。
  • 卖软件和服务： 单独卖软件授权，以及提供设备安装、维护、升级等技术服务【招股书 p.98】。

4. 客户是谁？

  • 主要是国内的半导体晶圆厂，比如客户一、客户二、芯联集成、奕瑞科技等【招股书 p.27】。也开始跟英飞凌等海外客户合作【招股书 p.27】。

二、赚不赚钱

核心结论： 公司收入增长很快，但利润波动巨大，2023和2024年都在亏钱，直到2025年才扭亏为盈。

1. 收入规模（快速扩张）

  • 2025年：3.93亿元【招股书 p.30】（大幅增长，近三年复合增长率50.25%【招股书 p.4】）

2. 利润情况（从巨亏到盈利）

  • 2023年：净亏损 -3,515.20万元【招股书 p.30】
  • 2024年：净亏损 -5,311.05万元【招股书 p.30】（亏损扩大）
  • 2025年：净利润 6,045.30万元【招股书 p.30】（扭亏为盈）

3. 毛利率（坐过山车）

  • 2023年：49.68%【招股书 p.22】
  • 2024年：24.67%【招股书 p.22】（暴跌）
  • 2025年：48.97%【招股书 p.22】（又涨回来了）
  • 解释： 公司自己说毛利率波动大，是因为不同产品的毛利率不一样，产品结构一变，整体毛利率就跟着变【招股书 p.22】。

4. 现金流（2023年很紧张）

  • 2023年：经营活动现金流净流出 -9,059.28万元【招股书 p.30】（花钱多，回款少）
  • 2024年：经营活动现金流净流入 9,583.77万元【招股书 p.30】（情况好转）
  • 2025年：经营活动现金流净流入 11,878.10万元【招股书 p.30】（造血能力增强）

三、过去怎么走的，未来怎么说的

1. 过去怎么走的（发展历程）

  • 2014年成立： 一开始做半导体设备的贸易生意【招股书 p.102】。
  • 2018年转型： 看到国内晶圆厂被日本公司垄断AMHS市场，决定自己搞研发【招股书 p.102】。
  • 2020-2022年： 从单点突破，先做出净化设备和存储设备，卖给国内大客户，同时把天车系统和软件也研发出来了【招股书 p.103】。
  • 2023-2025年： 开始能交付“整厂订单”了，在好几个晶圆厂完成了全套系统的验收，证明自己有整体交付能力【招股书 p.103】。

2. 未来怎么说的（募资计划）

  • 公司打算上市募资 11.91亿元，主要干四件事【招股书 p.31】：
  • 目标： 继续升级产品，攻克大型晶圆厂的天车系统市场，同时向东南亚、欧洲等海外市场扩张【招股书 p.4】。

四、行业什么样

1. 市场有多大？

  • 全球AMHS市场规模从2020年的21.36亿美元涨到2025年的40.53亿美元，复合增长率13.67%【招股书 p.121】。

2. 竞争格局（两座大山）

  • 极度垄断： 全球AMHS市场基本被两家日本公司——大福集团和村田机械瓜分，两家合计占了约90% 的市场份额【招股书 p.27】。
  • 国产替代机会： 因为贸易摩擦和供应链安全考虑，国内晶圆厂开始找国产供应商。弥费科技是少数能提供全套AMHS的国产公司，2025年全球市占率约1.60%，在中国大陆市占率约3.72%【招股书 p.124】。

3. 行业壁垒

  • 验证周期长： 晶圆厂不敢轻易换AMHS供应商，因为一旦出问题损失巨大。新产品需要经过长期、大规模的产线验证才能被接受【招股书 p.25】。
  • 技术集成难： 需要同时搞定机械、电气、软件、算法，还要保证系统稳定可靠，门槛很高【招股书 p.125】。

五、生意里的数字反差

1. 收入与利润的反差

2. 毛利率的反差

  • 2023年毛利率49.68%，2024年就跌到24.67%，2025年又回到48.97%。一年之内毛利率腰斩又翻倍，说明公司产品结构或定价策略变化很大，盈利模式不够稳定【招股书 p.22】。

3. 客户集中度的反差

  • 公司前五大客户收入占比，2023年是98.65%，2024年是80.64%，2025年是92.42%【招股书 p.21】。虽然2024年有所下降，但依然高度集中，公司对少数几个大客户的依赖非常严重。

4. 研发投入与亏损的反差

  • 公司2023年研发投入占收入22.18%，2024年占27.91%，2025年占13.47%【招股书 p.30】。2023和2024年亏钱的时候，研发投入比例反而更高，说明公司为了抢市场、搞技术，不惜代价在投入。

5. 存货与跌价准备的反差

  • 2025年底存货账面价值1.21亿元，但公司已经计提了2,616.41万元的存货跌价准备，占存货余额的17.77%【招股书 p.23】。这意味着公司手里有近两成的货可能卖不出好价钱，存在亏损风险。

六、值得关注的点 + 待核清单

值得关注的点

1. 扭亏为盈的可持续性： 2025年好不容易赚钱了，但之前两年都在亏。这个盈利是靠卖大项目一次性实现的，还是能持续下去？需要看未来几个季度的业绩。

2. 天车系统在大型晶圆厂的突破： 公司说天车系统已经在中小型厂验证通过，正在大客户的新建厂里测试【招股书 p.27】。这是公司未来增长的关键，如果测试顺利拿到大单，估值逻辑会完全不一样。

3. 客户集中度风险： 前五大客户占了九成收入，万一哪个大客户不买了，或者自己建厂计划推迟，对公司打击会非常大【招股书 p.21】。

4. 海外市场拓展： 公司计划在马来西亚建厂，目标是打入东南亚和欧洲市场【招股书 p.31】。能不能从日本巨头嘴里抢到海外订单，是检验公司真实竞争力的试金石。

5. 实控人夫妻持股： 实际控制人缪峰和柯娜是夫妻，加上一致行动人，合计控制公司约41.98% 的表决权【招股书 p.51】。公司治理的独立性需要关注。

待核清单

1. 客户身份： 招股书里提到的“客户一、客户二”等，具体是哪几家晶圆厂？[缺出处·待核]

2. 天车系统参数： 公司宣称天车系统“平均交付时间（ADT）压缩至110秒以内”【招股书 p.91】，这个指标在行业内属于什么水平？与日本竞争对手相比如何？[缺出处·待核]

3. 2024年毛利率暴跌原因： 招股书只说了“波动较为明显”，但没有详细解释2024年毛利率从49.68%跌到24.67%的具体原因。是哪个产品卖便宜了，还是成本涨了？[缺出处·待核]

4. 整厂订单的利润： 2025年整厂项目收入6,113.76万元，只占总收入的15.56%【招股书 p.98】。这些整厂项目毛利率是多少？会不会为了抢市场而牺牲利润？[缺出处·待核]

5. 员工持股平台： 公司通过上海弥正、上海弥真等平台搞了多层员工持股，结构比较复杂【招股书 p.52】。这些平台的股份支付费用未来还会不会继续影响利润？[缺出处·待核]
