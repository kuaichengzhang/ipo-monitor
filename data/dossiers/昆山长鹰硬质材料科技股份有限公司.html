一、这是个什么生意

长鹰硬科（简称“公司”）是一家做硬质合金的公司【招股书 p.14】。

1. 硬质合金是什么？

你可以把它想象成“工业牙齿”【招股书 p.15】。它是由碳化钨粉和钴粉等金属粉末，通过高温高压的粉末冶金工艺，烧制成的一种超级硬、超级耐磨的合金材料【招股书 p.12】【招股书 p.87】。它主要用来制作各种钻头、刀具、模具，去切削、打磨、钻孔其他金属或非金属材料【招股书 p.87】。

2. 主要卖什么？

公司主要卖两类东西：

  • 硬质合金（核心业务）：这是半成品，卖给下游的刀具、模具厂，让他们再加工成成品。比如棒材（做钻头、铣刀）、锯齿（做圆锯片）、模具合金（做冲压模具）等【招股书 p.88-89】。这块业务占公司2025年总收入的84.03%【招股书 p.89】。
  • 硬质合金工具（新业务）：这是成品，直接可以用的工具，比如数控刀片（用于数控机床）、工程截齿（用于矿山隧道挖掘）【招股书 p.89】。这块业务占2025年总收入的15.97%【招股书 p.89】。

3. 卖给谁？

下游客户很广，覆盖了装备制造、汽车、消费电子、航空航天、矿产能源等几乎所有工业领域【招股书 p.15】【招股书 p.87】。知名客户包括“山特维克”、“蓝帜工具”、“三一集团”等全球工具和模具品牌【招股书 p.15】。

4. 生意模式

  • 采购：主要买碳化钨粉和钴粉这两种原材料【招股书 p.12】。公司会根据订单和市场价格波动，适度囤一些货【招股书 p.89】。
  • 生产：接到订单后，按需生产，也会根据市场情况备一点货【招股书 p.89】。
  • 销售：大部分直接卖给终端大客户（占2025年收入的70.12%），小部分通过贸易商和经销商卖【招股书 p.89】【招股书 p.118】。

二、赚不赚钱

1. 收入与利润

  • 收入：2023年8.81亿元 → 2024年9.72亿元 → 2025年12.16亿元，持续增长【招股书 p.15】。
  • 净利润：2023年7,418万元 → 2024年6,360万元（下滑） → 2025年1.25亿元（大幅反弹）【招股书 p.15】。
  • 扣非后归母净利润：2023年6,811万元 → 2024年5,872万元（下滑） → 2025年1.20亿元（大幅反弹）【招股书 p.6】。

2. 毛利率

  • 主营业务毛利率：2023年21.79% → 2024年19.38%（下滑） → 2025年23.60%（回升）【招股书 p.6】。
  • 核心原因：毛利率和利润波动，主要是因为原材料（碳化钨粉）价格剧烈波动。2024年原材料涨价，但公司产品售价没跟上，导致利润下滑。2025年下半年原材料价格大涨，公司因为提前囤了低价原料，成本低，毛利率反而飙升【招股书 p.6】。

3. 赚钱能力（净资产收益率）

  • 扣非后加权平均净资产收益率：2023年12.39% → 2024年9.49%（下滑） → 2025年16.62%（大幅回升）【招股书 p.6】。说明公司为股东赚钱的能力在2025年显著增强。

4. 现金流

  • 经营活动现金流：2023年4,150万元 → 2024年2,305万元 → 2025年 -3.06亿元（大幅流出）【招股书 p.16】。
  • 原因：2025年原材料价格暴涨，公司为了锁定低价，大量囤货，导致现金大量支出【招股书 p.28】。这是“纸面富贵”的典型表现，利润很高，但钱都压在存货上了。

三、过去怎么走的，未来怎么说的

1. 过去怎么走的

  • 2023-2024年：收入增长，但利润下滑。原因是原材料涨价，但下游需求疲软，公司无法把成本完全转嫁给客户，导致毛利率下降【招股书 p.6】。
  • 2025年：收入、利润、毛利率全面爆发。主要原因是原材料价格单边快速上涨，公司凭借提前储备的低价库存，享受了巨大的成本优势，同时下游客户接受了产品涨价【招股书 p.6】【招股书 p.8】。
  • 2026年1-3月：业绩继续“狂飙”。营收5.62亿元，同比增长163.61%；扣非净利润1.66亿元，同比增长1429.35%【招股书 p.8】。毛利率更是飙升至44.20%【招股书 p.6】。

2. 未来怎么说的

  • 2026年1-6月业绩预计：公司自己预计营收12.15亿至13.43亿元，同比增长147.90%至174.00%；扣非净利润3.15亿至3.86亿元，同比增长933.93%至1,164.29%【招股书 p.9】。这个预测非常惊人。
  • 募资计划：计划募资3.48亿元，主要用来建一个年产1,800吨的高端硬质合金制品项目和研发中心【招股书 p.24】。项目建成后，每年会新增约2,011万元的折旧【招股书 p.8】。

四、行业什么样

1. 行业地位

  • 排名：根据中国钨业协会的统计数据，2022年至2024年，公司硬质合金产量的行业排名均位居国内第5名【招股书 p.14】。
  • 市场份额：2024年，公司产量占全国总产量的3.51%【招股书 p.23】。前四名分别是株洲硬质合金集团（13.58%）、厦门金鹭（9.53%）、自贡硬质合金（5.32%）、欧科亿（3.85%）【招股书 p.22-23】。

2. 行业特点

  • 上游：原材料是碳化钨粉和钴粉，价格波动剧烈，对行业利润影响巨大【招股书 p.5】。中国钨矿储量占全球52% 左右，产量占全球82.72%，有资源优势【招股书 p.99】。
  • 下游：应用极其广泛，是“工业耗材”，需求受宏观经济影响大【招股书 p.5】。下游客户对价格敏感度相对较低（因为硬质合金工具只占其产品成本1%-5%）【招股书 p.8】。
  • 竞争格局：行业集中度不高，前十大厂商产量合计占全国50.04%【招股书 p.115】。低端产品同质化竞争严重，高端产品与国际先进水平仍有差距【招股书 p.115】。

五、生意里的数字反差

1. 利润与现金流的反差：2025年公司净利润1.25亿元，但经营活动现金流却是 -3.06亿元。赚了钱，但钱没进口袋，全变成了存货【招股书 p.15】【招股书 p.16】。

2. 毛利率的剧烈波动：2024年主营业务毛利率19.38%，2025年23.60%，到了2026年1-3月更是飙升至44.20%【招股书 p.6】。这种波动完全取决于原材料价格和公司囤货策略，不是常态。

3. “真实”与“账面”利润的巨大差距：公司自己测算，如果没有低价库存的优势，按市场价格算，2026年1-3月的毛利率只有10.31%，毛利5,794万元，净利润1,316万元。但公司实际实现的毛利率是44.20%，毛利24,837万元，净利润16,747万元。账面利润是“真实”利润的12.7倍【招股书 p.6】。

4. 存货激增：2024年末存货账面价值2.48亿元，2025年末飙升至5.95亿元，一年翻了一倍多【招股书 p.7】。这既是公司锁定低价原料的“功劳”，也是未来如果原料价格下跌，可能产生巨额跌价准备的“隐患”。

5. 应收账款与信用期外占比：2025年末应收账款余额3.18亿元，其中信用期外的占比高达28.38%【招股书 p.7】。这意味着有近三成的客户没有按时付款。

六、值得关注的点 + 待核清单

值得关注的点

1. 业绩的“含金量”：公司2025年及2026年一季度的爆发式增长，核心驱动力是原材料价格单边上涨和低价库存。这种盈利模式高度依赖对原材料价格的精准判断和运气。一旦原材料价格停止上涨甚至下跌，公司的利润和毛利率将面临巨大压力。

2. 存货的“达摩克利斯之剑”：公司账上囤了价值5.95亿元的存货。如果未来碳化钨价格大幅下跌，公司将面临计提大额存货跌价准备的风险，这可能会瞬间吞噬掉所有利润【招股书 p.7】。

3. 现金流压力：经营活动现金流已经为负，且公司还在大规模投资建厂（募投项目）。如果未来原材料价格回落，公司不仅利润会受影响，还可能面临巨大的资金周转压力。

4. 对赌协议风险：公司实际控制人与部分股东签有对赌协议，如果上市失败，回购权将恢复，可能导致实际控制人持股比例发生变化【招股书 p.7】。

5. 客户集中度与供应商竞争：前五大材料供应商采购占比高达60.70%【招股书 p.27】，且部分供应商（如厦门钨业、章源钨业）本身就是公司的竞争对手，存在供应链被卡脖子的风险【招股书 p.27】。
